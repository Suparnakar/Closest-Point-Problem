There are 3 files.
Main_file.java, Point.java, MergeSort.java all have to be in same folder same package called kar.suparna under src.

This program can be run in Eclipse, IntelliJ Idea, or running from command line.

For command line compilation:
javac *.java
cd ..
cd ..


For command line run the code:
java kar.suparna.Main_file
